# NovaMart Retail Cloud Platform

## Client

NovaMart Electronics (Simulated)

## Business Problem

The company stores customer, product and sales data inside Excel spreadsheets.

There is no centralized database.

There are no backups.

Reporting is manual.

## Objectives

- Centralize data
- Build ETL pipelines
- Deploy using Docker
- Store backups in AWS S3
- Automate infrastructure using Terraform

## Success Criteria

- Production-ready architecture
- Secure cloud storage
- Automated deployments
- Analytics dashboard
